Netflix Data Analysis

Dataset: Netflix
https://www.kaggle.com/datasets/imtkaggleteam/netflix

About:
This project explores the Netflix Movies and TV Shows dataset to uncover trends in content types, genres, release years, countries of origin, and ratings. It includes data cleaning, transformation, and exploratory data analysis using Python libraries like pandas, matplotlib, and seaborn. The goal is to derive meaningful insights and visualize patterns in Netflix's global content catalog, laying the foundation for future enhancements like interactive dashboards and recommendation systems.
